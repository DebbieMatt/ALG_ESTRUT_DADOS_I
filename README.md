## 🎯 Sobre o Projeto

Este repositório reúne os exercícios resolvidos ao longo da disciplina de Algoritmos e Estrutura de Dados I. Os programas cobrem desde operações aritméticas básicas até estruturas condicionais e de repetição, com foco no aprendizado progressivo da linguagem C.

> ⚠️ **Nota:** Por se tratar de um curso introdutório, o objetivo principal é o aprendizado dos fundamentos — boas práticas avançadas foram deixadas em segundo plano intencionalmente.

## ▶️ Como Compilar e Executar

### Pré-requisitos

```bash
gcc --version
```

### Clonar o repositório

```bash
git clone https://github.com/DebbieMatt/ALG_ESTRUT_DADOS_I.git
cd ALG_ESTRUT_DADOS_I
```

### Compilar e executar

```bash
gcc -Wall -o saida nome_do_arquivo.c
./saida
```
